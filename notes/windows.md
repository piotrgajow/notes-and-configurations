# JQ on Windows using Git Bash

- [Download jq for Windows](https://stedolan.github.io/jq/download/)
- Put the executable somewhere (e.g. `~/tools/jq-win64.exe`)
- Add alias to `~/.bashrc` (e.g. `alias jq=~/tools/jq-win64.exe`)
- Enjoy!
