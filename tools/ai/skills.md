## AI Skills

Skills are specific prompts, between system and user prompts. Skills describe how to apply knowledge to perform an
action or solve a problem. Any context/prompt that generates good responses can be converted into a reusable skill.

### Custom skills

| Name                                       | Usage  | Description                                                               | 
|--------------------------------------------|--------|---------------------------------------------------------------------------|
